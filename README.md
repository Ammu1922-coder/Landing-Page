# MainCrafts Landing Page

## Overview
MainCrafts is a modern and responsive landing page designed to showcase services and features of a digital business.  
The page emphasizes clean design, usability, and a professional look, making it suitable for startups, small businesses, or personal portfolios.

## Features
- **Fast**: Optimized for quick loading and smooth performance.  
- **Responsive**: Works seamlessly across desktop, tablet, and mobile devices.  
- **Scalable**: Built with flexibility in mind to easily expand or customize.

## Technologies Used
- **HTML5** for structured content.  
- **CSS3** for styling, including responsive layouts and hover effects.  
- **Font Awesome** for icons.  
- **Google Fonts (Poppins)** for a clean and modern typography.  

## Layout
The landing page contains three main sections:
1. **Header & Navigation**: Sticky header with dropdown menu for services.  
2. **Hero Section**: Eye-catching call-to-action with heading, description, and button.  
3. **Features Section**: Highlights the key aspects of the business in interactive cards.  
4. **Footer**: Contains copyright information and links.
